# 🎓 RUET CGPA Calculator

![HTML](https://img.shields.io/badge/HTML-5-orange?style=for-the-badge)
![CSS](https://img.shields.io/badge/CSS-3-blue?style=for-the-badge)
![JavaScript](https://img.shields.io/badge/JavaScript-ES6-yellow?style=for-the-badge)
![Status](https://img.shields.io/badge/Project-Live-success?style=for-the-badge)

A **modern and mobile-friendly CGPA Calculator** built according to the grading system of **Rajshahi University of Engineering & Technology (RUET)**.

This tool allows students to **enter marks for each course**, automatically converts them into **letter grades**, and calculates the **final CGPA using credit-weighted calculation**.

# 📊 RUET Grading System

| Marks    | Grade | Grade Point |
| -------- | ----- | ----------- |
| 80 – 100 | A+    | 4.00        |
| 75 – 79  | A     | 3.75        |
| 70 – 74  | A-    | 3.50        |
| 65 – 69  | B+    | 3.25        |
| 60 – 64  | B     | 3.00        |
| 55 – 59  | B-    | 2.75        |
| 50 – 54  | C+    | 2.50        |
| 45 – 49  | C     | 2.25        |
| 40 – 44  | D     | 2.00        |
| < 40     | F     | 0.00        |
